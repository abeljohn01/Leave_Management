# Leave Management (Frontend-only)

Angular 17+ front-end demo for a Leave Management System using Angular Material, Reactive Forms and a simple shared service for state.

Run:

1. Install dependencies

```bash
cd leave-management
npm install
```

2. Start dev server

```bash
npx ng serve --open
```

Notes:
- No backend required. State persists only in memory.
- Mock employees are in `src/app/services/leave.service.ts`.
