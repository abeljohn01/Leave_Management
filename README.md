# Leave Management (Frontend-only)

Angular 17+ front-end demo for a Leave Management System using Angular Material, Reactive Forms and a simple shared service for state.

Run:

1. Install dependencies

```bash
cd leave-management
npm install
```

2. Start dev server

```bash
npx ng serve --open
```

Notes:
- No backend required. State persists only in memory.
- Mock employees are in `src/app/services/leave.service.ts`.
- Deploy with Vercel by linking this repository to a new Vercel project.

Vercel deployment steps:
1. Push the `leave-management` folder to a GitHub repository.
2. In Vercel, import the repo and create a new project.
3. Vercel will run `npm install` and `npm run build` automatically.
4. Your public URL will be generated by Vercel after deployment.
